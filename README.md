# Deezweb

Web application named Deezweb, user can search musique, add to favorites, remove, listen, etc ...

# Technologies

- HTML
- CSS
- Javascript
- jQuery
- Bootstrap

# How to test

1. Go to current folder of project in terminal
2. Run "live-server"
3. Server will be launched for me is (http://127.0.0.1:8080/)
4. Go to your url
5. Enjoy !
